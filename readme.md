# 💪 Fitness Tracker Application

A full-stack **Fitness Tracking Web Application** built using **Java Spring Boot** and **Thymeleaf** that allows users to manage workouts, track calorie intake, and monitor fitness goals in a simple and effective way.

🌐 **Live Application**  
👉 http://fantastic-journey-production-cee3.up.railway.app/

---

## 🧠 Overview

This application is designed to help users stay consistent with their fitness journey by providing tools to:

- Track workouts 🏋️
- Monitor calorie intake 🔥
- Set and achieve goals 🎯
- Visualize progress 📊

The project follows a **layered architecture** and demonstrates core backend development skills using Spring Boot.

---

## ✨ Key Features

### 🏋️ Workout Management
- Add, update, and delete workouts
- Track sets, reps, and duration

### 🔥 Calorie Tracking
- Log daily calorie intake
- Monitor consumption patterns

### 🎯 Goal Setting
- Define personal fitness goals
- Track progress over time

### 📊 Dashboard
- Centralized view of fitness data
- Quick insights into activity

### 🌐 Server-Side Rendering
- Dynamic UI using Thymeleaf templates

---

## 🏗️ Architecture

The project follows a **layered architecture pattern**:



### 🔹 Layers Explained

- **Controller Layer**
  - Handles HTTP requests and responses

- **Service Layer**
  - Contains business logic

- **Repository Layer**
  - Interacts with the database

- **View Layer**
  - Thymeleaf templates for UI rendering
