# 💪 Fitness Tracker Web App

A full-stack fitness tracking application built using **Java Spring Boot** and **Thymeleaf** that helps users log workouts, set fitness goals, and track calorie intake efficiently.

🌐 **Live Demo:**  
👉 http://fantastic-journey-production-cee3.up.railway.app/

---

## 🚀 Features

- 🏋️‍♂️ **Workout Tracking**
  - Add and manage exercises (sets, reps, duration)
  
- 🎯 **Goal Management**
  - Set fitness goals and monitor progress
  
- 🔥 **Calorie Tracking**
  - Track daily calorie intake
  
- 📊 **Dashboard**
  - Visual overview of fitness activity
  
- 🌐 **Server-Side Rendering**
  - Built using Thymeleaf for dynamic UI rendering
